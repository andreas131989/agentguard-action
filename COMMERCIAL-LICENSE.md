# AgentGuard Commercial License

Commercial licensing is not active yet.

AgentGuard is currently in public beta.

## Planned commercial direction

The intended commercial model is:

- public/open-source repository usage remains free or low-friction
- private commercial repository usage may require a paid license
- agency/client work may require a paid license
- team usage may be licensed by repository count or team size

## Public beta

During public beta, commercial/private repository usage is handled manually.

## No license server

AgentGuard does not currently use:

- hosted license validation
- telemetry
- repo verification
- usage tracking
- source code collection
- account creation
- billing integration

The `license-key` action input is reserved for future commercial beta usage.

In the current public beta, it is optional and not validated.

## Expected future plans

Possible future plans may include:

| Plan | Intended use |
| --- | --- |

| Team | Multiple private repositories |
| Pro | Larger teams or advanced presets |

These plans are not active yet.

## Product boundary

AgentGuard does not:

- prove code is safe
- prove code is correct
- perform security scanning
- perform AI code review
- replace human reviewers
- block merges by default

AgentGuard flags pull requests that may deserve extra human attention.
